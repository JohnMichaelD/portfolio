#### What I Learned:

While building this portfolio site, I jumped into learning SASS, and other tools like Font Awesome. 

Building this portfolio site was a great way to understand the benefit to CMS-like builds. By using SASS and tools like Font Awesome, I can make quick and accurate changes to my website across multiple pages and code. This will be extremely helpful for when I start larger and larger projects!

The build out of this portfolio website also reinforced the importance of how one breaks down attributes for their HTML elements.
